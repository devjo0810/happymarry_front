## node 확인
nodejs 있을 경우 불필요

nvm 설치(setup) https://github.com/coreybutler/nvm-windows/releases

nvm install v14.15.4

nvm use 14.15.4

node -v (노드 버전 확인)

npm -v (npm 버전 확인)

nvm ls (nvm 목록 확인)

## 프로젝트 설정 방법
### npm 사용시
```
npm install
```
### yarn 사용시
```
yarn install
```

## 개발서버 실행 방법
### npm 사용시
```
npm run dev
```
### yarn 사용시
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
